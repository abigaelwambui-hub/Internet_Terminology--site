DNS The Internets Phone book

DNS (Domain Name System) makes the internet human-friendly by translating domain names into IP addresses. Instead of memorizing long numeric addresses, users type names like www.example.com, and DNS resolves them into machine-readable IPs so computers know where to connect. This system works like a smartphone’s contact list: you search by name, and the phone retrieves the number automatically. DNS records include A (IPv4), AAAA (IPv6), CNAME (aliases), and MX (mail servers). Without DNS, users would need to remember complex IP addresses, making navigation impractical. It is a hierarchical, distributed system that ensures scalability and speed in resolving billions of queries daily.
